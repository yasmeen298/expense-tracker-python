# expense-tracker-python
An expense tracker made from basic python and fundamentals
This is a console based expense tracker covering basic python topics; made for educational purpose to practice my python skills
THIS EXPENSE TRACKER allows user to keep track of their expenses by adding expenses and getting summaries of their total expenses.

### My Technical Improvements
**Language refractoring** Independently translated the original Hinglish code to English to ensure it is readablle by developers worldwide.
**Code Optimization**     Analyzed the original script and removed some features.I chose to focus the app on core data entry and individual                            tracking for a leaner user experience.
**Enhanced Robustness**   Added a critical feature that checks if the user enters alphabets instead of numbers in the amount.
HOW IT WORKS: If a user types "one hundred fifty nine" instead of "159", the app now displays "INVALID INPUT! PLEASE ENTER A NUMBER FOR AMOUNT.
**Comprehensive Debugging & Code Audit**  I performed an extensive audit of the codebase to resolve critical runtime issues. This included Identifying and fixing errors like NameError and Indentation Errors that hindered the applications stability.
